# QuizMint AI

> *"Your AI-powered quiz generation companion – affordable, intelligent, and built with passion."*

A lightweight, AI-driven webapp for instant quiz generation, created by **John Dakultor** as a self project. Features a sleek subscription system with Stripe, PayPal, and GPay, wrapped in an elegant ATM‑style modal.

---

##  Features

-**AI‑Powered Quiz Generation** – Generate quizzes from any topic, text input, or youtube url.
-**Integrated Payments** – Stripe, PayPal, and Google Pay.
-**Fully Responsive** – Works beautifully on mobile, tablet, and desktop.
-**Dynamic Currency Detection** – Auto‑detects user location and converts prices.
-**Secure & PCI‑Compliant** – All payments handled via certified gateways.
-**Clean, Modern UI** – Built with Tailwind CSS, shadcn/ui, and Lucide icons.

---
